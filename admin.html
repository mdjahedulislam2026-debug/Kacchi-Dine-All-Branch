Firebase Realtime Database Rules

যদি PERMISSION_DENIED error আসে, তাহলে নিচের Rules Firebase Console-এ পেস্ট করুন:
Firebase Console → Realtime Database → Rules ট্যাব → নিচের কোড পেস্ট → Publish 

{
  "rules": {
    "settings": {
      ".read": true,
      ".write": "auth != null"
    },
    "menu": {
      ".read": true,
      ".write": "auth != null"
    },
    "packages": {
      ".read": true,
      ".write": "auth != null"
    },
    "orders": {
      ".read": "auth != null",
      ".write": true
    },
    "zones": {
      ".read": true,
      ".write": "auth != null"
    }
  }
}
📌 auth != null মানে — শুধু লগইন করা অ্যাডমিন লিখতে পারবেন
📌 orders .write: true মানে — কাস্টমার অর্ডার দিতে পারবেন (লগইন ছাড়া)
📌 সব .read: true মানে — কাস্টমার মেনু দেখতে পারবেন
